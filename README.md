## Quick Start Guide

### For CLI Projects (Swapper & Arbitrage Bot)

1. **Fork this repository** (click "Fork" button above)
2. **Clone your fork:**
```bash
   git clone https://github.com/YOUR_USERNAME/jupiter-iaue-hackathon-2025.git
   cd jupiter-iaue-hackathon-2025
```
3. **Choose a project:**
```bash
   cd cli-swapper
   # OR
   cd arbitrage-bot
```
4. **Install dependencies:**
```bash
   npm install
```
5. **Run it:**
```bash
   npm start
```

### For Whale Watcher

1. **Fork this repository**
2. **Open `whale-watcher/index.html`** in your browser
3. **Start coding!**

No installation needed - runs directly in browser!
